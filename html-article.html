HTML Tutorials

Introduction to HTML

HTML (HyperText Markup Language) structures the content of web pages. Here’s an example:

<!DOCTYPE html>
<html>
<head>
  <title>HTML Example</title>
</head>
<body>
  <h1>Welcome to HTML!</h1>
  <p>This is a paragraph.</p>
  <a href="https://example.com">Visit Example</a>
</body>
</html>

Creating a Basic HTML Page

Here’s how to create a page with headings, images, and links:

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>My Web Page</h1>
  <p>Learn more about HTML below.</p>
  <img src="image.jpg" alt="A descriptive image">
  <a href="#more">Read more</a>
</body>
</html>

CSS Tutorials

CSS Basics

Use CSS to style HTML elements. Here’s an example of inline, internal, and external styles:

<style>
  body {
      font-family: Arial, sans-serif;
  }
  .highlight {
      color: red;
      font-weight: bold;
  }
</style>

<body>
  <p class="highlight">This is styled text!</p>
</body>

CSS Grid Example

CSS Grid is powerful for layouts:

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}
.item {
  background: #0073e6;
  color: white;
  padding: 20px;
  text-align: center;
}

Flexbox Example

Flexbox aligns items efficiently:

.container {
  display: flex;
  justify-content: space-around;
  align-items: center;
  height: 200px;
}
.item {
  background: #ff5733;
  color: white;
  padding: 10px;
  border-radius: 5px;
}

JavaScript Tutorials

Adding JavaScript to HTML

Include JavaScript in your HTML using the <script> tag:

<script>
  console.log('Hello, JavaScript!');
</script>

Interactive JavaScript Example

Here's a basic interactive button example:

<button id="myButton">Click Me!</button>

<script>
  document.getElementById('myButton').addEventListener('click', function() {
      alert('Button clicked!');
  });
</script>
